# bsd_video

This repo contains the python+manim+sage scripts used
to create the video "What is the Birch and
Swinnerton-Dyer conjecture?" now available at

https://youtu.be/dkHvP5-VFXE

The video differs significantly from the version that
can be compiled here. The material here was 
split up and given to a professional local
video creation company bottletop.co.uk.
They added the nice animated character, voice over etc.
